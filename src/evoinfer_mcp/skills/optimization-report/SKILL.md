# Optimization Report

Use this skill when summarizing an inference optimization task.

Report structure:
- Objective and scope.
- Environment and workload.
- Baseline benchmark and evidence artifacts.
- Bottleneck diagnosis and profiler evidence.
- Candidate attempts, including rejected attempts.
- Before/after benchmark table.
- Correctness validation result.
- Promotion or rejection decision.
- Risks, missing evidence, and next steps.

Output expected from this skill:
- Concise engineering report.
- Artifact list.
- Honest conclusion that distinguishes verified gains from tentative observations.
